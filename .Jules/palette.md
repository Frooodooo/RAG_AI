## 2024-10-27 - Icon Button Accessibility
**Learning:** Found several icon-only buttons (`md-copy-btn` in `ChatMessage.tsx`, Rename, and Clear in `ChatPage.tsx`) that lacked `aria-label`s. Visual hover feedback was partially present via `title` but it wasn't standardized across all interactive icons, missing from the copy code button entirely. The Cosmos Dark theme handles styling effectively, but semantic meaning for screen readers is easily missed on inline-styled icon buttons in this codebase.
**Action:** Always verify `aria-label` and `title` attributes exist on any `<button>` that doesn't contain text nodes, especially inside inline-styled or complex React components like those in the Chat interface.
